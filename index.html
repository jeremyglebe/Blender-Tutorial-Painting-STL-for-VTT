Blender Tutorial: STL Miniatures to VTT Character Art

A practical workflow for taking an STL miniature, adding simple materials, lighting it, applying subtle outline/shadow effects, and exporting it as usable VTT portrait/token art.

1. Goal and Workflow

Goal: Use STL minis as fast character-art sources for a VTT.

Not the goal: Perfect 3D texturing, realistic rendering, or deep Blender mastery.

Main idea:

  1. Import STL
  2. Assign broad material colors
  3. Set an orthographic camera
  4. Use Sun + Area Light
  5. Add Ambient Occlusion for recess shading
  6. Add subtle Freestyle lines
  7. Render to transparent PNG
  8. Do minor cleanup in an image editor
This workflow works well because the STL provides the pose and sculpt detail. Blender provides the color, lighting, and stylization needed to turn it into usable art.

2. Tools and Names

What I kept calling it Proper name Shortcut / Menu Use
C brush Circle Select C Manual brush selection. Great cleanup tool. Probably the main one.
L thing Select Linked Hover + L Only good for truly separate mesh islands.
linked flat faces Linked Flat Faces Select → Select All by Trait → Linked Flat Faces Smart surface selection by angle. Extremely useful.
hide Hide Selected H Hide current selection so you can work behind/in it.
unhide Reveal Hidden Alt + H Bring hidden faces back.
invert Invert Selection Ctrl + I Select everything except the current selection.

Recommended selection workflow

  1. Select one or a few faces.
  2. Use Linked Flat Faces.
  3. Adjust the angle/degrees setting until it grabs the right surface.
  4. Use Circle Select to clean up.
  5. Use H / Alt + H as needed.

In practice: Linked Flat Faces + Circle Select is the core combo.

Optional: make Linked Flat Faces easier to access

  • Right-click the menu item and choose Assign Shortcut.
  • Or right-click and choose Add to Quick Favorites, then access it with Q.

3. Import and Material Painting

Import

  1. File → Import → STL
  2. Select the STL.
  3. Press Home if you need to frame the model.

Enter the correct mode

  • Select the mini.
  • Press Tab for Edit Mode.
  • Press 3 for Face Select.
  • Switch viewport shading to Material Preview.

Assign materials

  1. Select the faces you want.
  2. Open Material Properties.
  3. Click + to add a material slot.
  4. Click New.
  5. Set the color/material properties.
  6. Click Assign.
Material Type Suggested Settings Notes
Cloth Metallic 0, Roughness 0.6–0.9 Use for robes, tunics, pants, banners.
Leather Metallic 0, Roughness 0.5–0.8 Boots, belts, straps, pouches.
Metal Metallic 1.0, Roughness 0.25–0.5 Armor, weapons, buckles.
Bone / Horn Metallic 0, Roughness 0.5–0.8 Skulls, tusks, horns, claws.
Skin / Fur Metallic 0, Roughness 0.5–0.8 Broad tones are enough.
Do not obsess over tiny details. Broad readable material zones matter more than perfect precision.

If the mesh is too low-detail for face-based painting

To add more faces in one area:

  1. Select the faces.
  2. Right-click → Subdivide.
  3. Adjust Number of Cuts.

Important: use small numbers. Try 1–4 cuts first.

Do not accidentally type 100 cuts unless you want to visit the Warp.

4. Camera Setup

Why orthographic?

An orthographic camera removes perspective distortion. That makes the mini feel less like a photographed toy and more like game art.

Set up the camera

  1. Add a camera if needed: Shift + A → Camera
  2. Frame the model in the viewport.
  3. Press Ctrl + Alt + Numpad 0 to snap the camera to your current view.
  4. Press Numpad 0 to look through the camera.
  5. Select the camera.
  6. Open Camera Properties.
  7. Set Type → Orthographic.
  8. Adjust Orthographic Scale for zoom/framing.

Move camera using normal controls

  1. Enter camera view with Numpad 0.
  2. Press N.
  3. Open the View tab.
  4. Enable Camera to View.
  5. Now move the view normally to reposition the camera.
  6. Disable Camera to View when done.

5. Lighting

Keep it simple. Use two lights:

Sun Light = broad visibility Area Light = focused portrait light

Sun Light

  1. Shift + A → Light → Sun
  2. Set strength roughly around 0.3–1.5.

Use the Sun to keep the model readable. Lower it if the image looks too flat.

Area Light

  1. Shift + A → Light → Area
  2. Select it and press G to move it.
  3. Use G then X, Y, or Z if needed.
  4. Place it in front, above, and slightly to one side.
  5. Rotate with R if desired.
Area Light Setting Suggested Value Effect
Power 300–800 Main light intensity
Size 3–6 Larger = softer shadows
Your discovered setup is valid: Sun + Area Light works well. The Sun acts like general illumination, and the Area Light gives a focused portrait effect.

6. Ambient Occlusion

What it is

Ambient Occlusion (AO) darkens creases, folds, contact points, and recessed areas. Think: digital recess shading / wash.

Use Eevee / Eevee Next

  1. Open Render Properties.
  2. Set Render Engine to Eevee or Eevee Next.
  3. Switch the viewport to Rendered View.

Where AO is in newer Blender

Usually under:

Render Properties → Raytracing → Fast GI Approximation → Method → Ambient Occlusion

What to adjust

Setting What it does What to do
Distance How far the recess shading spreads Main setting to tune
Resolution / Rays / Steps / Precision Quality settings Adjust later if needed

How to tune AO

  1. Look through the orthographic camera.
  2. Adjust AO Distance while watching the result.
  3. Increase it until recesses become visible.
  4. Stop before the whole model becomes muddy.

Scale matters

AO settings depend heavily on model scale. A huge model may need much larger values than a normal infantry-sized or hero-sized model.

Personal note from prior testing: on a very large giant-sized mini, the AO effect became more noticeable around 50 distance and could go higher for darker recesses.

If AO seems subtle

  • That is normal.
  • AO is meant to work with lighting, not replace it.
  • Lower Sun strength if lighting is flattening the effect.
  • Judge AO from the final camera angle, not just from random viewport angles.

7. Linework / Outline

Use Freestyle

  1. Open Render Properties.
  2. Enable Freestyle.
  3. Render with F12 to see the result.

Line settings

  • Open View Layer Properties for Freestyle settings.
  • Use line thickness 1.
Thickness 1 is the sweet spot. Higher than that tends to look intentionally cartoonish.
Freestyle Setting Recommendation
Thickness 1
Edge types Start with Silhouette and Contour
Extra detail Optionally add Crease if needed

8. Export Settings

Transparent background

  1. Open Render Properties.
  2. Open Film.
  3. Enable Transparent.

Output format

  1. Open Output Properties.
  2. Set file format to PNG.
  3. Set color to RGBA.

Recommended render sizes

Use Suggested Size
Portrait / token source 1024 × 1024
Full-body character art 1200 × 1600 or larger
Large creature 1600 × 2000 or larger

Render

  1. Press F12.
  2. When done: Image → Save As.
  3. Save as PNG.

9. Post-Processing

Do final cleanup in an image editor like GIMP, Krita, Photoshop, Photopea, etc.

  1. Crop tighter so the character reads clearly.
  2. Add subtle outer glow or shadow to separate the figure from backgrounds.
  3. Add a thin outer stroke if needed for readability.
  4. Slightly boost contrast if the image feels flat.
  5. Downscale if needed for final VTT use.
Keep post effects subtle. The goal is polish, not turning the image into a sticker.

10. Reusable Template

Once you get a scene working well, save a reusable Blender file.

Template should include

  • Orthographic camera
  • Sun Light
  • Area Light
  • Eevee / Eevee Next
  • AO enabled
  • Freestyle enabled
  • Transparent background
  • Common material slots

Suggested filename:

vtt-stl-render-template.blend

Future mini workflow

  1. Open the template.
  2. Import STL.
  3. Paint material zones.
  4. Adjust camera framing.
  5. Tune AO for model scale.
  6. Render and export.

11. Troubleshooting

Problem Likely fix
Material color is not visible Use Material Preview or Rendered View.
Linked Flat Faces selects almost nothing Increase the angle/degrees setting.
Linked Flat Faces selects too much Lower the angle/degrees setting, then clean up with Circle Select.
Select Linked grabs the whole model The STL is fused. Use Linked Flat Faces + Circle Select instead.
Dragging the Area Light just resizes it Select it, then move with G.
Lighting looks flat Lower Sun strength or move Area Light more to the side.
AO seems too subtle Increase Distance; judge through the camera; lower Sun strength.
AO makes the whole model muddy Lower Distance.
Freestyle lines look too cartoony Use thickness 1.
Background is not transparent Enable Film → Transparent and export PNG RGBA.
Blender freezes after massive subdivide Try Ctrl + Z. If needed, reopen and recover/autosave. Use smaller subdivide values next time.
