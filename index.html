A practical, forgiving workflow

Turn an STL into art that feels drawn.

A beginner-friendly Blender guide for giving collected miniatures readable color, controlled light, and a clean VTT-ready silhouette—without needing to become a professional 3D artist first.

Begin the guide ↓ Jump to the recipe

The guiding idea: the STL already solves pose and sculpt detail. The task is to make important shapes readable at the size used in play.

A useful mental model

Think like a painter with a camera.

The goal is not a perfect digital miniature. The process arranges a handful of visual signals so the character reads quickly in a scene.

Silhouette

Does the pose and outer shape identify the character? If not, fix the camera before adding detail.

Color blocks

Cloth, leather, metal, skin, and props should separate at a glance. Broad zones do the heavy lifting.

Depth cues

Do folds and contact points separate? Lighting and AO should support the sculpt, not bury it.

STL pose + broad materials + orthographic framing + selective depth = usable VTT art
Good stopping rule: if the render reads at the size of an in-game token, more detail may be polish—not progress.
1
Lesson one · prepare

Get the mesh ready to work.

Import, frame, and inspect before assigning colors. A clean setup prevents confusion later.

Import and inspect

  1. Open a fresh scene. Keep the default camera and lights only when their purpose is clear.
  2. Import the STL. Use File → Import → STL; the exact menu label can vary with the Blender version.
  3. Frame the model. Select it and press Home if it disappears in the viewport.

Check these three things

  • Scale: does the object appear absurdly huge or tiny compared with the grid?
  • Orientation: which side is the character's front?
  • Geometry: are there floating parts, holes, or unexpected loose pieces?
Do not fix everything yet. Note problems that affect the camera or materials; leave cosmetic cleanup for later.
Checkpoint: the model should be easy to orbit around, with its front and main shapes clearly identifiable before entering Edit Mode.
2
Lesson two · paint

Paint broad zones, not tiny pixels.

STLs are often one fused mesh. That is fine. Different materials can still be assigned to faces selected by their surface angle and cleaned up manually.

The core selection loop

  1. Choose the object and press Tab for Edit Mode.
  2. Press 3 for Face Select. Use Material Preview so colors are visible.
  3. Select a few faces on the target surface.
  4. Use Select → Select All by Trait → Linked Flat Faces. Adjust its angle until it catches the right zone.
  5. Use C for Circle Select to add or remove faces. Hover over a connected island and press L only when the geometry is truly separate.

Hide and recover

Painting the back of a model is easier when the front is temporarily out of the way.

  • H hides the current selection.
  • Alt + H reveals hidden geometry.
  • Ctrl + I inverts the selection.

If Linked Flat Faces grabs too much, lower the angle. If it grabs too little, raise it. Circle Select is the cleanup tool, not a failure state.

Start With a Primer Material

Before painting individual parts, give the whole model a base material. This is the digital equivalent of priming a real miniature.

The first material added to a Blender object often becomes the material used by the whole object. If the entire model changes color, that is normal and does not necessarily indicate an incorrect selection.

  • Name: Primer or Primer (Matte Black)
  • Color: very dark gray, almost black
  • Surface: Diffuse / matte
  • Metallic: 0 · Roughness: high, if available

Treat it as the unpainted/default color. Then add skin, cloth, metal, leather, and other materials and assign them to selected faces.

A useful starter palette
ZoneStarter materialWhat matters visually
ClothMetallic 0 · Roughness .6–.9Matte, large color shapes
LeatherMetallic 0 · Roughness .5–.8A slightly deeper value than cloth
MetalMetallic 1 · Roughness .25–.5Value contrast and a few highlights
Skin / furMetallic 0 · Roughness .5–.8Enough contrast to separate it from clothing
Bone / hornMetallic 0 · Roughness .5–.8A light or warm accent zone
When the mesh is too low-detail: select only the needed area, right-click → Subdivide, and start with 1–4 cuts. Subdivision is a local tool. A huge cut count can make Blender unresponsive.
Checkpoint: switch to Material Preview and squint at the model. If the large color relationships make sense, painting can stop for now.
3
Lesson three · frame

Make the camera do the graphic design.

An orthographic camera keeps parallel lines parallel and removes the “photographed action figure” feeling of perspective.

Set the camera

  1. Add a camera with Shift + A → Camera if needed.
  2. Compose the pose in the viewport, then use Ctrl + Alt + Numpad 0 to align the active camera to the active view.
  3. Press Numpad 0 to look through the camera.
  4. In Camera Properties, change Type to Orthographic.
  5. Adjust Orthographic Scale for the crop. Smaller scale means a closer framing.

Compose for a token

  • Leave a little breathing room around the silhouette.
  • Keep the face and the main prop out of the extreme edge of the frame.
  • Try a slight three-quarter angle instead of a perfectly flat front view.
  • Check the crop at the final output size, not only while zoomed into the viewport.
Camera first: if the character does not read as a silhouette, lighting and materials cannot rescue the composition.
Easy camera movement: in camera view, open the N panel → View → enable Camera to View. Move the view normally, then disable it when framing is finished.
4
Lesson four · light

Use two lights with different jobs.

Start simple. A broad Sun keeps the model readable; an Area Light creates a directional portrait feel.

Sun = broad visibility

Add Shift + A → Light → Sun. Start around 0.3–1.5 strength and lower it if the render becomes flat.

A Sun's position does not matter the same way an Area Light's position does; its rotation controls the direction of the light.

Area = portrait shape

Add Shift + A → Light → Area. Place it in front, above, and slightly to one side. Move with G; rotate with R.

Start around 300–800 power and size 3–6, then tune by eye. Larger size gives softer shadows.

Too flat? Lower Sun strengthToo harsh? Increase Area sizeFace too dark? Move Area toward cameraDepth missing? Move Area farther to the side
Checkpoint: Rendered View should show a readable face, a light side, and a shadow side. The target is shape—not drama.
5
Lesson five · depth

Let recesses and edges explain the sculpt.

Ambient Occlusion gives folds and contact points a little extra separation. Freestyle can add a clean graphic boundary.

Ambient Occlusion (AO)

Use Eevee / Eevee Next for a fast feedback loop. In newer layouts, AO may appear under a Raytracing or Fast GI Approximation area in Render Properties.

  1. Look through the final camera.
  2. Increase AO Distance until creases become visible.
  3. Stop before the whole model turns muddy.

AO depends on scene scale. Large-giant tests around 50 are useful reference points, not universal settings.

Freestyle linework

Enable Freestyle in Render Properties and render with F12. Start with a line thickness of 1.

  • Begin with Silhouette and Contour.
  • Add Crease only when the sculpt needs more help.
  • If it looks too cartoonish, lower line weight before changing everything else.
Order matters: tune lighting first, then AO, then Freestyle. Otherwise the source of a visual change remains unclear.
6
Lesson six · export

Export a clean VTT source.

Keep the background transparent so the same render can work on different maps, cards, tokens, and portraits.

Transparent PNG

  1. Render Properties → Film → enable Transparent.
  2. Output Properties → set file format to PNG.
  3. Set color to RGBA, not RGB.
  4. Press F12, then Image → Save As.

Good starting sizes

UseSize
Square token source1024 × 1024
Full-body portrait1200 × 1600

Render larger when practical, then downscale. Keep the transparent master for future crops.

Post-processing pass
Restraint is a feature: post-processing should improve readability, not turn the miniature into a sticker.
7
Lesson seven · repeat

Save the boring parts once.

When one scene works, turn it into a template. The next session can focus on the character—not rebuilding the lighting rig.

Keep in the template

  • Orthographic camera and a reliable framing reference
  • Sun + Area Light with sensible starting values
  • Eevee / Eevee Next render setup
  • AO and Freestyle defaults
  • Transparent film and PNG / RGBA output
  • Common material slots and a simple palette

The short version next time

  1. Open vtt-stl-render-template.blend.
  2. Import the STL and check orientation.
  3. Assign broad material zones.
  4. Frame the orthographic camera.
  5. Tune AO for the model's scale.
  6. Render, save the transparent PNG, and make a smaller VTT copy.
Template habit: save a new working file per character. Keep the template pristine; it is the known-good starting point.
When something looks wrong

Troubleshoot by visual symptom.

Change one variable at a time. The fastest fix is usually the control that has the same visual job as the problem.

The color is not visible.
Use Material Preview or Rendered View. Confirm the material was assigned to the selected faces—not only created in a slot.
Adding a material changed the whole model's color.
This is normal for the first / base material. Treat it as the model's Primer material, then add new materials for selected face regions.
Linked Flat Faces selects almost nothing.
Raise the angle / degrees setting, then use Circle Select to fill in the edges.
Linked Flat Faces selects too much.
Lower the angle / degrees setting. Hide geometry or use Circle Select for cleanup.
Select Linked grabs the entire miniature.
The STL is probably one fused island. Use Linked Flat Faces instead; Select Linked is only useful for truly separate islands.
Dragging the Area Light resizes it.
Select the light and use G to move it. Use S only when a size change is intended.
The lighting looks flat.
Lower Sun strength, move the Area Light farther to one side, or reduce fill from the front.
AO is too subtle.
Increase Distance, judge through the camera, and make sure the Sun is not washing out every recess.
AO makes the model muddy.
Lower Distance. AO is an accent, not the main source of shading.
Freestyle looks too cartoonish.
Start with thickness 1 and only Silhouette / Contour. Add more line types only when the problem they solve is clear.
The background is not transparent.
Enable Film → Transparent and export PNG with RGBA. Verify the checkerboard appears in the render result.
Blender freezes after subdividing.
Try Ctrl + Z or recover from the autosave. Next time, subdivide a small selection with 1–4 cuts.
Keep experimenting

Field notebook

The best settings are those that make sense in the context of the model, camera, and intended token size. Record experiments instead of treating a single number as law.

Known good

  • Freestyle thickness 1 is a strong starting point.
  • Sun + Area Light is simple and reliable.
  • AO should be judged through the final camera.
  • Linked Flat Faces + Circle Select is the core selection combo.

Worth testing

  • Warm vs. cool key light
  • Flat-color materials vs. slightly varied roughness
  • Silhouette-only linework for very small tokens
  • Different orthographic crops for portrait and token use

Write down

  • Blender version and render engine
  • Model scale or imported dimensions
  • AO distance that helped
  • Output size and where the render was used
Version note: Blender moves panels and renames render features over time. If a menu path differs, search the relevant Properties editor for the feature name—Ambient Occlusion, Freestyle, Transparent, or Orthographic—then record the path in notes.md.
Optional upgrades: Keep this basic flow intact and explore the separate Advanced Techniques library for smarter materials, complex lighting, backgrounds, kitbashing, compositing, or reusable templates.
