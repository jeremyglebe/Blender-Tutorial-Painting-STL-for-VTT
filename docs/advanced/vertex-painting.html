Advanced technique 05

Paint variation onto the mesh.

Vertex painting adds color information to vertices instead of creating a new material zone for every mark. It is fast, expressive, and limited by the mesh density.

← Advanced OverviewPaint a test mark ↓
Purpose

Add marks without UV setup.

Use it for

Scars, dirt near boots, a dark cloak hem, beard variation, bruising, mutation marks, or green stains near runes.

Know the limit

Color lives at vertices / faces. A low-density STL cannot hold delicate marks; the paint will look blocky.

Required context

Object selected, Vertex Paint mode, a brush, and a material that actually reads the vertex color attribute.

Where it fits: after the basic material zones and before final render. It is a lighter setup than texture painting, but more limited.
Step by step

Paint a small, visible test first.

  1. Select the object in Object Mode. If the mesh is extremely coarse, consider a small subdivision or remesh first.
  2. Switch the mode dropdown to Vertex Paint. Choose a brush color and start with a medium brush size and strength around .25–.5.
  3. Paint a red scar, dark boot dirt, or green rune stain. Use F to adjust brush size where supported, and adjust strength in the tool settings.
  4. Switch colors from the color picker. Lower strength for blush, grime, or blended variation; raise it for a graphic mark.
  5. Return to Material Preview / Rendered View. If nothing changes, connect the color attribute to the material's Base Color in the Shader Editor.
  6. Save the .blend. If your Blender version offers an explicit color attribute or image-data save action, save that data too.

How it differs

  • Material assignment: whole selected faces use a material.
  • Vertex paint: color is interpolated across the mesh.
  • Texture paint: color is stored in an image mapped through UVs.
Make color visible

The shader must read the paint.

Quick check

  1. Open Shader Editor.
  2. Add or locate the color attribute / vertex color node for your Blender version.
  3. Connect its color output to Base Color, or mix it gently with the existing material color.
  4. Preview through the same camera you will render.

Common mistakes

  • Painting a coarse mesh and expecting fine lines.
  • Painting successfully but using a material that ignores the color attribute.
  • Using full strength everywhere, producing stickers instead of variation.
  • Forgetting to save the blend file after painting.
VTT rule: if a mark is important enough to identify the character, make it large enough to survive downscaling.
Next choice

Need finer marks?

Move to Texture Painting when vertices cannot hold the detail you want. Stay with Vertex Paint when speed and broad variation matter more than perfect control.
