How to use this library

Nothing here is required.

Finish one basic render first. Then choose the page that solves the next visible problem. Each page is self-contained and identifies its place in the original process.

Start with one upgrade

Opening every node editor at once is unnecessary. Pick the page that addresses the visible weakness in the current render.

Keep the house style

Advanced does not mean busier. At VTT scale, silhouette, face, weapon, and major color zones still win.

Save experiments separately

Keep a known-good basic scene. Try advanced ideas in a copy for easy comparison.

Recommended progression

A gentle order of operations.

01

Smarter Materials

Make cloth, leather, metal, bone, skin, hair, and glow feel less flat.

02

Complex Lighting

Build key, fill, rim, mood, and emissive focal points.

03

Backgrounds

Add a simple stage that supports the silhouette instead of competing with it.

04

Post Processing

Use Blender-first polish for contrast, glow, shadow, and final sizing.

05

Vertex Painting

Add localized color variation without setting up UVs.

06

Texture Painting

Move into detailed marks, patterns, tattoos, and heraldry.

07

Cel / Toon Shading

Push the look toward comic, illustrated, or graphic rendering.

08

Rendering & Compositing

Separate passes so AO, shadows, emission, and lines can be tuned later.

09

Camera & Kitbash

Make character-specific variants from existing STL parts—no sculpting required.

10

Creating a Template

Turn the best decisions into a repeatable campaign house style.

+

Alternate Tools

Optional companions when a specialized tool makes one step less frustrating.
