Advanced technique 06

Paint details with an image texture.

Texture painting gives you more detail than vertex colors, but it asks for more setup: UVs, an image texture, a material connection, and a careful save routine.

← Advanced OverviewSet up a texture ↓
Purpose

Use pixels when vertices are not enough.

Use it for

Facial markings, tattoos, battle damage, clothing patterns, heraldry, warpaint, and controlled gradients.

Know the cost

STLs often have no useful UVs. Unwrapping, image setup, and external saving create more failure points.

Required context

Edit Mode for UVs, Shader Editor for the material, and Texture Paint mode for painting.

Save warning: save the .blend and save the image texture externally. Do not assume your paint is safe just because the Blender file is saved.
Step by step

Build the smallest useful setup.

  1. Select the STL and enter Edit Mode. Select all faces with A.
  2. Open the UV Editing workspace. Use U → Smart UV Project as a practical first unwrap. Inspect for extreme stretching.
  3. In the Shader Editor, create an Image Texture node and choose New. Start with a square image such as 2048 × 2048 if the character needs detail.
  4. Connect the Image Texture color to Base Color, or mix it with the basic material color so you can still control the surface.
  5. Switch to Texture Paint. Select the image texture as the paint slot and paint one obvious test mark.
  6. Use the Image Editor's Image → Save As to save the painted image outside the blend file. Then save the blend.

Why UVs matter

UVs are the map that tells Blender which image pixels belong on which mesh surfaces. Without a useful map, texture paint has nowhere predictable to go.

Avoid lost work

Use a two-save habit.

Save the image

Save the painted PNG or other image texture to a known folder after meaningful passes.

Save the blend

Save the Blender scene after the material, UV, and texture connections change.

Pack later

If you plan to move the project, use Blender's packing option only after the external image is already safe.

Common mistakes: painting the wrong image, forgetting to connect it to the material, relying on automatic packing, or trying to paint a tiny detail on a heavily stretched UV island.
VTT check

Detail is only helpful when it survives.

Use Texture Paint for marks that read as identity at token size. If a tattoo becomes a gray smudge, enlarge it, simplify it, or return to broad material zones.

Next: if the detail is good but the overall style still feels too realistic, try Cel / Toon Shading.
