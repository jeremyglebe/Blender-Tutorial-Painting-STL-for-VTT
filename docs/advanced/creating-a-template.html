Advanced technique 10

Turn good decisions into a house style.

A template keeps camera, lights, output, materials, and background choices ready. The next character can begin with art direction—not a blank scene.

← Advanced OverviewBuild the template ↓
Purpose

Save the boring parts once.

Use it when

A known-good render, repeated character work, or a campaign needs consistent art across PCs, NPCs, and monsters.

Keep adjustable

Model, camera crop, AO distance, light power, background choice, and any character-specific materials.

Required context

Collections, Outliner, save-as workflow, render settings, and enough confidence to name things clearly.

Where it fits: after the basic workflow—or after an advanced scene has proven useful. Do not template an untested setup.
Build the scene

Create a calm, labeled workspace.

  1. Open the known-good character scene and remove the old model. Keep the working camera, lights, materials, and output settings.
  2. Create Collections named MODEL, LIGHTS, CAMERA, BACKGROUNDS, and MATERIALS_REFERENCES.
  3. Rename important objects: CAM_ORTHO, KEY_SUN, KEY_AREA, and optional RIM_LIGHT.
  4. Keep Eevee / Eevee Next, AO, Freestyle, transparent Film, PNG / RGBA, and a useful output resolution.
  5. Save the file as basic-vtt-render.blend. Then duplicate it for each new character.
  6. Import a new STL into the MODEL collection, replace the old model, and save a new version before painting.

Starter variants

  • basic-vtt-render.blend
  • dark-fantasy-render.blend
  • comic-lines-render.blend
  • magic-glow-render.blend
  • portrait-token-render.blend
Template checklist

Include the repeatable decisions.

Rendering

Orthographic camera, Eevee setup, AO defaults, Freestyle, transparent film, PNG / RGBA, output resolution.

Art direction

Common material slots, background options, light colors, line weight, and any campaign palette.

Safety

Duplicate for new work. Save versions. Keep the original template pristine and easy to return to.

Common mistake: locking every value so the template becomes a cage. Templates should remove setup work, not remove judgment.
VTT check

Consistency is the payoff.

A campaign house style makes a simple STL library feel intentional. Keep the same framing, line weight, and contrast logic while letting each character's materials and kitbash carry identity.

Final optional stop: if Blender becomes frustrating for one specialized step, see Alternate Tools. The core workflow can still stay in Blender.
