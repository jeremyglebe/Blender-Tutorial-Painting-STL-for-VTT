Advanced technique 07

Push the render toward illustration.

Cel / toon shading turns continuous light into clearer bands. It pairs naturally with Freestyle, but it is easy to overdo—start with a small stylistic nudge.

← Advanced OverviewCompare approaches ↓
Purpose

Make the lighting language more graphic.

Use it when

Comic, illustrated, or campaign-specific art benefits from clear light bands and stronger edge language.

Do not bother when

The basic render already has the desired style. Full toon materials can make sculpted miniatures look artificial if the campaign does not support it.

Required context

Material Properties and Shader Editor. Freestyle from the basic workflow remains useful. Eevee is a practical test engine.

Where it fits: after basic materials and lighting; use it before final Freestyle / compositor tuning.
Four levels

Choose the least complex approach that works.

1 · Freestyle + flat materials

The easiest entry point. Keep broad materials, use a controlled key light, and set Freestyle thickness to 1. This may already be enough.

2 · Toon BSDF

In the Shader Editor, add a Toon BSDF if the documented Blender version provides it, then connect it to Material Output. Start with a soft size and low roughness change.

3 · ColorRamp bands

Use a diffuse or shader value as the Fac input of a ColorRamp. Set two or three stops to create hard light bands, then mix with the existing material.

4 · Lighting-driven style

Use harder shadows, a strong rim, and a limited color palette. This changes the image without asking every material to become a toon shader.

Simple node pass

Try a two-band material.

  1. Duplicate one basic material so the original stays safe.
  2. In the Shader Editor, add a Toon BSDF or a diffuse-to-ColorRamp path.
  3. Use a dark color for the shadow band and the base color for the light band.
  4. Keep the transition readable; a giant node graph is unnecessary for a first test.
  5. Render with Freestyle thickness 1, then compare at token size.

Conservative start

Two light bands, one key light, one rim, and one line weight. If the result feels too artificial, soften the light or return one material to Principled shading.

Common mistake: using a hard ramp on every material, every light, and every edge until the character becomes a pile of unrelated stripes.
VTT check

Graphic does not mean noisy.

Readability win

Clear bands can help armor, hair, and folds separate when a soft render disappears at small scale.

Style risk

Too much toon influence can make miniatures look goofy or cheap. Let the campaign style decide.
