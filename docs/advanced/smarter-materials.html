Advanced technique 01

Make materials feel intentional.

Broad color zones are enough to start. This page adds controlled roughness, variation, bump, hair experiments, and emission so the render feels less like a flat plastic model.

← Advanced OverviewStart the material pass ↓
Purpose

Use surface response to do some of the painting.

A good material tells the viewer “cloth,” “leather,” “metal,” or “hair” before they inspect the sculpt. You can get that difference with a few values instead of manually painting every highlight.

Use it when

Your colors read, but every zone has the same plastic feel—or hair, bone, metal, and leather need more separation.

Do not bother when

The token is tiny, the silhouette is unclear, or the basic lighting is not working yet. Fix the larger signal first.

Required context

Object Mode for material assignment; Material Properties and Shader Editor for deeper control. Eevee is a fast place to test.

Where it fits: use this after the basic material zones and before final lighting / render tuning.
Step by step

Start with the surface setting.

  1. Select the object and open Material Properties. Choose the material slot you want to improve.
  2. Inspect the material's Surface choice. A Principled BSDF is a strong default, but other surface or shader presets can change the result dramatically.
  3. Try small changes to Base Color, Roughness, and Metallic. Render a test after each meaningful change.
  4. For a controlled variation, open the Shader Editor, add Noise Texture, pass it through a ColorRamp, and mix it gently with the base color or roughness.
  5. For eyes or runes, use an emission surface or connect an Emission shader. Start low; let lighting and compositing do the rest.

The hair discovery

Some surfaces behave very differently. A Hair-style surface or shader made red hair / mohawks look much better in a successful render than a plain Principled material.

Treat Hair as an experiment, not a universal answer. It may be the wrong fit for a chunky sculpt or a graphic style.

Starting settings

Useful defaults, not laws.

MaterialStarting pointFirst thing to adjust
MetalMetallic 1.0 · Roughness .25–.55Roughness: lower for sharper highlights
LeatherMetallic 0 · Roughness .55–.85Small color noise or a slightly darker value
Bone / hornMetallic 0 · Roughness .5–.8Warm off-white / tan variation
Hair / furTry Hair-style surface if available · moderate roughnessColor variation and key-light direction
Eyes / runesEmission color of choice · low to moderate strengthStrength, then bloom / glare if needed
Noise rule: if you can see the procedural pattern as “noise” at token size, it is probably too strong. Variation should break up a flat color, not become the subject.
VTT check

Judge the material at the final scale.

Readability

Metal should not become a black blob. Hair should keep its silhouette. A glow should point to the face or prop.

Contrast

Different roughness values are helpful, but value and hue differences still matter more after downscaling.

Common mistake

Adding nodes before checking the light. A material can appear wrong because the light is wrong.
