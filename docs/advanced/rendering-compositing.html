Advanced technique 08

Keep the render in pieces.

Render passes let you tune parts of the image after the first render. This is a real step up in complexity, so use it when the basic compositor pass cannot give you enough control.

← Advanced OverviewReview the passes ↓
Purpose

Separate the signals you keep adjusting.

Use it when

You want stronger AO, softer shadows, quieter lines, brighter runes, or isolated materials without rerendering every decision.

Do not bother when

You are still learning the basic render. Passes add setup and can make a simple job feel like a pipeline.

Required context

View Layer Properties, Render Layers, the Compositor, and basic node connections. Pass availability varies by engine and Blender version.

Where it fits: after the basic render is stable; keep the clean Combined / Beauty render as your fallback.
Pass map

Know what each pass is for.

PassUse it for
Combined / BeautyThe normal finished render and your comparison baseline.
Ambient OcclusionStrengthen or soften recess shading separately.
ShadowAdjust the darkness and contribution of cast shadows.
DiffuseControl base non-shiny surface contribution.
Glossy / SpecularControl reflections, highlights, and metallic shine.
EmissionMake eyes, runes, and magic brighter or softer.

Transparent AlphaKeep a clean character cutout over a transparent background.
Mist / DepthAdd atmospheric fade or depth-based effects.
NormalAdvanced relighting or stylized effects.
Cryptomatte / Object / Material IDIsolate a hair, weapon, armor, material, or object for adjustment.
Beginner node route

Make one controlled compositor change.

  1. Open View Layer Properties and enable only the passes you expect to use. Names and locations can vary by engine.
  2. Open the Compositing workspace and enable Use Nodes.
  3. Use the Render Layers node as the source. Send the Combined output to Composite so the normal render still works.
  4. Add a Mix / Multiply or Color Balance node for one pass, such as AO or Emission.
  5. Compare the change against the Combined render. Lower the mix before you add another branch.
  6. Connect the result to Composite and optionally Viewer. Save the blend and the final output.

Useful first experiments

  • Multiply AO at low opacity
  • Lower Freestyle line opacity
  • Boost Emission before adding glare
  • Darken only the Shadow branch
  • Use Mist behind a character
  • Isolate hair or weapon with an ID pass
Common mistake: enabling every pass, building a giant node graph, and losing the ability to tell which change helped.
VTT check

Compositing should buy readability.

If the final token is harder to read, return to the Combined render. Preserve transparent alpha and keep a clean export for maps that already provide the atmosphere.
